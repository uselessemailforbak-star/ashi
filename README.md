# I'm Sorry! - Apology Website

A romantic, interactive apology website with animations, background music, and floating hearts.

## Features

- 🎨 Beautiful pink-themed UI with smooth animations
- 💖 Floating hearts background animation
- 🎵 Background music support (plays on first interaction)
- ✨ Interactive buttons with visual feedback
- 📱 Fully responsive design
- 🎬 Pulsing profile image animation
- 🎉 Confetti effect on "Yes" button

## Files

- `index.html` - Main HTML file with embedded CSS and JavaScript
- `audio.mp3` - Background music (currently referenced as audio.mp3)
- `photo.jpg` - Profile image (needs to be added)

## Deployment

### Deploy to Vercel
```bash
npm i -g vercel
vercel
```

### Deploy to Netlify
```bash
npm i -g netlify-cli
netlify deploy
```

### Deploy to Heroku
```bash
heroku create
git push heroku main
```

### Local Development
```bash
npm install
npm start
```

Then visit `http://localhost:8000` in your browser.

## Notes

- Replace `photo.jpg` with your actual image file
- Update `audio.mp3` if you want different background music
- The site works on all modern browsers
- No backend required - purely static HTML, CSS, and JavaScript
